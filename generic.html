Comprehensive COVID-19 Analysis

Using HTML & Python

This project looks at various COVID-19 metrics on a county-by-county level in order to gauge what pockets of the United States are at the highest risk. All data is recorded and visualized real-time by querying from the New York Times' GitHub repository and the chief visualization include COVID Cases by County, Elderly Count by County and Average COVID-19 Rate of Change by State. Each visulization is dynamic (changing as cases increase/decrease). Furthermore, I also perform a novel clustering algoranalysis of each and every county in the United States according to some user-specified metrics. The results of clustering are visualized and can be downloaded (as a CSV) for further analysis/research by the user. (Click "Full Story" to access the live web application)
